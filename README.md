# Simics Device Source Code

This repository contains device source code from various Simics packages.

## Source

The packages were downloaded and installed from:
https://lemcenter.intel.com/productDownload/?Product=256660e5-a404-4390-b436-f64324d94959

## Directory Structure

The following device directories have been extracted and organized by package:

- `simics-7.57.0/src/devices` - Core Simics devices
- `simics-qsp-x86-7.38.0/src/devices` - Quick Start Platform x86 devices
- `simics-risc-v-simple-7.12.0/src/devices` - RISC-V Simple platform devices
- `simics-training-7.0.0-pre.11/src/devices` - Training package devices

Each directory maintains the original hierarchical structure from the Simics installation.
